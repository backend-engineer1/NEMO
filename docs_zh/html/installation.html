如何安装¶

Requirements

  1. Python 3.6 or 3.7

  2. CUDA >= 10.0

  3. cuDNN >= 7.6

  4. APEX

  5. PyTorch >=1.2

  6. (Recommended for distributed training) NCCL >= 2.4

Tip

Instead of installing all requirements. They are all automatically included in the NVIDIA’s PyTorch container . You can pull it like so: docker pull nvcr.io/nvidia/pytorch:19.08-py3

Installing NeMo and Collections

  1. Clone the repository:

git clone https://github.com/NVIDIA/nemo

  1. Go to nemo folder and do: python setup.py install

  2. Install collections

    1. ASR collection from collections/nemo_asr do: sudo apt-get install libsndfile1 && python setup.py install

    2. NLP collection from collections/nemo_nlp do: python setup.py install

    3. LPR collection from collections/nemo_simple_gan do: python setup.py install

For development do: python setup.py develop instead of python setup.py install

  1. Run unittests from the nemo directory to validate installation:

python -m unittest tests/*.py

All tests should pass without errors.

  1. Go to examples/start_here to get started with few simple examples
