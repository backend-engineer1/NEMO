Welcome to NEMO!¶

NEural MOdules (NeMo) is a high level toolkit for training AI applications using Neural Modules. NeMo comes with neural module collections for automatic speech recognition (ASR) and natural language processing (NLP).

NeMo is built for fast training on GPUs. It provides:

  • Mixed-precision training using Tensor Cores on NVIDIA’s Volta and Turing GPUs

  • Multi-GPU and multi-node distributed training

  • Distributed evaluation
